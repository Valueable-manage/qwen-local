# Qwen 问答助手 - 项目流程分析

## 一、入口总览

| 入口 | 用途 |
|------|------|
| `deploy.bat` | 首次部署：安装依赖 + 下载模型 + 启动 |
| `run.bat` | 日常启动：直接运行（需已部署） |
| `main.py` | 命令行对话：纯 CLI，不启动 Web |

---

## 二、部署流程（deploy.bat）

```
deploy.bat
    └── setup.ps1
            ├── 1. 检查/安装 Python 3.12+
            ├── 2. 检查/安装 uv
            ├── 3. 检查/安装 Node.js
            ├── 4. 检查/安装 VC++ Redistributable（PyTorch 依赖）
            ├── 5. uv sync（Python 依赖）
            ├── 6. npm install（前端依赖）
            ├── 7. 下载模型（可选，Qwen3.5-4B ~8–10GB）
            ├── 8. 创建桌面快捷方式 Qwen.lnk
            └── 9. 启动应用（uv run python start.py）
```

- `deploy_cn.bat`：设置 `USE_MODELSCOPE=1` 后调用 `deploy.bat`，使用国内镜像

---

## 三、运行流程（run.bat / 桌面快捷方式）

```
run.bat
    ├── venv 存在 → .venv\Scripts\python.exe start.py
    └── venv 不存在 → uv run --no-sync python start.py
            ├── 1. 启动 FastAPI 后端（uv run python server.py）
            │       └── lifespan: load_model() 加载 Qwen
            ├── 2. 等待 2 秒
            ├── 3. 启动 Next.js 前端（npm run dev）
            ├── 4. 轮询检测 localhost:3000 就绪（最多 30 秒）
            └── 5. 自动打开浏览器
```

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  用户: 双击 run.bat 或 Qwen.lnk → uv run python start.py                   │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                    ┌───────────────┴───────────────┐
                    ▼                               ▼
        ┌───────────────────┐           ┌───────────────────┐
        │  FastAPI 后端      │           │  Next.js 前端     │
        │  server.py         │           │  frontend/        │
        │  :8000             │           │  :3000           │
        └─────────┬─────────┘           └─────────┬─────────┘
                  │                               │
                  │  OpenAI 兼容 API              │  CopilotKit UI
                  │  /v1/chat/completions         │  用户对话
                  │  /v1/models                   │
                  │  /api/cancel                  │
                  │  /api/shutdown                │
                  │                               │
                  └───────────────┬───────────────┘
                                  ▼
                    ┌─────────────────────────────┐
                    │  model_loader.py            │
                    │  - load_model() 加载 Qwen   │
                    │  - chat() 非流式            │
                    │  - chat_stream() 流式       │
                    └─────────────────────────────┘
```

## 五、后端：server.py

| 接口 | 方法 | 作用 |
|------|------|------|
| `/v1/models` | GET | OpenAI 兼容，列出模型 |
| `/v1/chat/completions` | POST | 对话接口，支持 stream |
| `/api/cancel` | POST | 中断当前生成 |
| `/api/shutdown` | POST | 关闭整个程序 |
| `/` | GET | 重定向到 localhost:3000 |

- **lifespan**：启动时 `load_model()`，退出时清理
- **流式**：`req.stream=True` 时用 `chat_stream()`，返回 SSE

---

## 六、模型加载：model_loader.py

```
load_model()
    ├── torch.cuda.is_available() ?
    │   ├── 是 → GPU：4-bit 量化 或 float16
    │   └── 否 → CPU：float32
    ├── 读取 qwen/Qwen3___5-4B（或 MODEL_PATH）
    └── 缓存 _tokenizer、_model
```

---

## 七、前端：frontend/

| 文件 | 作用 |
|------|------|
| `app/page.tsx` | 主页面，CopilotChat，页签关闭时 sendBeacon shutdown |
| `app/api/copilotkit/route.ts` | CopilotKit Runtime，转发到 127.0.0.1:8000/v1 |
| `app/api/cancel/route.ts` | 代理 → FastAPI /api/cancel |
| `app/api/shutdown/route.ts` | 代理 → FastAPI /api/shutdown |

- CopilotKit 使用 OpenAI 兼容接口，`baseURL: http://127.0.0.1:8000/v1`

---

## 八、对话数据流

```
用户输入
    → CopilotKit 请求 /api/copilotkit
    → route.ts 用 OpenAI SDK 调用 http://127.0.0.1:8000/v1/chat/completions
    → server.py 解析 messages，调用 chat_stream()
    → model_loader 流式生成 token
    → SSE 流式返回前端
    → CopilotChat 逐字显示
```

**中断**：用户点停止 → `/api/cancel` → `_stop_flag[0]=True` → StoppingCriteria 在下个 token 停止

---

## 九、关闭流程

| 方式 | 触发 | 结果 |
|------|------|------|
| 关闭页签 | `beforeunload` → `sendBeacon("/api/shutdown")` | 后端 `os._exit(0)`，start.py cleanup 杀前端 |
| Ctrl+C | start.py 捕获 | cleanup 杀前后端 |

---

## 十、可选入口：main.py

```
uv run python main.py
```

- 纯命令行对话，不启动 Web
- 直接调用 `model_loader.chat()`，与 Web 流程独立
